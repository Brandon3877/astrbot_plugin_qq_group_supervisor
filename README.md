# QQ群监督员

QQ群监督员是一个用于群员消息审核的AstrBot插件，结合LLM分析群员消息，可以自动执行群管行为。

## 详细介绍

QQ群监督员是一个用于群员消息审核的AstrBot插件。它可以根据可配置的筛选器收集群消息，并将消息打包发给LLM，执行基于语境和用户所设置群规的合规程度评估。评估同时可以根据用户提前配置的群管理行为选项，例如撤回、警告、禁言等，交由LLM选择。这些行为均由用户在WebUI提前配置，LLM不能自己创造未配置的行为。用户可以根据设置，选择自动执行或询问执行。用户在配置一个插件管理员后，插件将在群规范被违反或氛围不和谐时，把氛围评估和选择的操作，以聊天记录形式发送给插件管理员，将执行群撤回的消息也会被包含在聊天记录内。

## 使用方法

请在WebUI进行配置，未来可追加在消息平台更改配置的功能。

## 工作原理

1. 对于配置内启用的群聊，按群聊为单位，分别记录新的群消息
2. 当消息达到一定条数，或达到时间限制时，发送给LLM评估
3. LLM结合语境评估群员行为，根据用户配置的规范，判断是否合规
4. 根据插件用户提前配置的群管行为，由LLM从中做出选择
5. LLM返回结果后，插件将结果转为便于阅读的格式，发给插件管理员
6. 插件根据设置，自动执行群管操作，或等待插件管理员确认后执行
